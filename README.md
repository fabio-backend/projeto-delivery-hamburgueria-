# projeto-delivery-hamburgueria-
Projeto de um Sistema de Delivery para Hamburgueria, desenvolvido para automatizar pedidos online. O sistema permite ao cliente escolher produtos, personalizar lanches, pagar e acompanhar o pedido. Para a empresa, oferece controle de cardápio, pedidos, entregas e relatórios, otimizando o atendimento e as vendas.

Descrição do Projeto – Sistema de Delivery para Hamburgueria
O projeto consiste no desenvolvimento de um Sistema de Delivery para uma Hamburgueria, com o objetivo de automatizar e otimizar todo o processo de pedidos, desde a escolha do lanche pelo cliente até a entrega final.

A plataforma permitirá que os clientes realizem pedidos online de forma rápida, prática e intuitiva, escolhendo produtos do cardápio, personalizando hambúrgueres, selecionando formas de pagamento e acompanhando o status do pedido em tempo real.

Para a hamburgueria, o sistema oferecerá um painel administrativo completo, possibilitando o gerenciamento de:

Cardápio (produtos, preços, adicionais e promoções).

Pedidos em tempo real.

Clientes cadastrados.

Entregas e motoboys.

Formas de pagamento.

Relatórios de vendas e desempenho.

O sistema também contará com controle de status do pedido, como:
Recebido → Em preparo → Pronto → Saiu para entrega → Entregue, garantindo melhor organização e comunicação entre cozinha, atendimento e entrega.
